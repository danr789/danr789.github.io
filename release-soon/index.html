Toontown Toonline Getting Ready for Developer Alpha Release!

Posted on September 12, 2021 by Prof. Robin
Alpha Release

Hello There Fellow Toons! I am Prof.Robin, the Lead Developer at Toontown Toonline. What is Toontown Toonline you may ask? Good Question, Toontown Toonline is a new Toontown server whose main purpose is have a new Storyline, new Cogs, and a new Boss, Plus other Good Stuff ( Calm down, you won't see ALL those feature any time soon), while still keeping nostalgic features such as Python Injectors available for you all to mess around!

Great! When Can I be able to Play It?

Hmm, probabaly when Cartoonival Starts. By the Way cause this is Still Alpha, we will not exactly be online any time soon.

I am Sure you are asking ' Why can't we play it Online '. Well the answer is: Firstly, the code still has bugs meaning it can crash anytime if you do certain stuff, and Secondly, we might make it online when the TTT reaches to Beta.

Until Then I am Prof. Robin. Good Bye!
